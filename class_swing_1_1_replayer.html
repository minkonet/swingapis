Replay Swing.Replayer.currentReplay
get

CWS, Current Working Replay

It is null if there is no current working replay. To Record and Load set this value.

float Swing.Replayer.currentTime
get

Current replay time

  • In Recording mode, this is the elaspsed time since the beginning of recording.
  • In PlayingBack mode, this is the time offset of the current playback position from the beginning of a replay file.
  • In Broadcasting mode, this is the elapsed time since the beginning of broadcasting.
  • In Watching mode, this is the time offset of the current watching position from the beginning of broadcasting.
float Swing.Replayer.endTime
get

Time length of a replay media

  • In Recording mode, this is the same as currentTime.
  • In PlayingBack mode, this is the time length of a replay file.
  • In Broadcasting mode, this is the same as currentTime.
  • In Watching mode, this is the same as currentTime.
